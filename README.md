# 🌟 NeonPortfolio 

Portofolio pribadi bertema **hitam, kuning neon, dan hijau neon** yang dibangun murni dengan **HTML dan CSS (Flexbox)**.

---

## 🎯 Fitur Utama

- ✅ **Semua elemen HTML wajib**: `<header>`, `<nav>`, `<footer>`, `<main>`, `<article>`, `<aside>`  
- ✅ **Anchor di `<nav>`** menuju profil Dicoding (silakan ganti dengan username Anda)  
- ✅ **Foto di `<aside>`** (dapat diganti dengan foto diri) + biodata  
- ✅ **Layout Flexbox 100%** (tidak menggunakan float)  
- ✅ **Tombol aksi**: About Me, Kontak, Pendidikan, LinkedIn, GitHub  
- ✅ **Responsif** – support mobile & tablet (media query)  
- ✅ **Satu `<h1>`** dan heading terstruktur dengan baik  
- ✅ **Semua gambar memiliki `alt` text**  
- ✅ **CSS terpisah** dari HTML (eksternal)  

---

## 🛠️ Teknologi yang Digunakan

| Teknologi | Fungsi |
|-----------|--------|
| HTML5 | Struktur semantik halaman |
| CSS3 | Styling, Flexbox, animasi neon, media query |
| Google Fonts | Font Inter modern |
| (Opsional) Gambar eksternal | Foto profil dari placehold.co / randomuser.me |
